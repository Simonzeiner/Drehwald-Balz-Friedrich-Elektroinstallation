# Drehwald-Balz Elektrounternehmen

Premium-Website für Friedrich Drehwald-Balz Elektrounternehmen in Mainz-Gonsenheim.

## Inhalt

- moderne Startseite mit Leistungen, Ablauf, Kundenstimmen und Kontakt
- separate Seiten für Impressum und Datenschutz
- lokale Originalbilder der bestehenden Website
- keine Analyse-Tools, keine Marketing-Cookies, kein eingebettetes Kartenmaterial
- strukturierte Daten für lokale Suchmaschinenoptimierung

## Entwicklung

```bash
pnpm install
pnpm run dev
pnpm run build
pnpm test
```

## Projektstruktur

- `app/`: Seiten, Layout und Styling
- `public/original/`: übernommene Bilder und Inhaltsreferenzen der bestehenden Website
- `public/og.png`: Linkvorschau-Grafik
- `.openai/hosting.json`: Hosting-Konfiguration
